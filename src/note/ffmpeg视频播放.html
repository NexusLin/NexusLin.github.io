ffmpeg的使用

1、认识和安装ffmpeg

-官网 https://ffmpeg.org/download.html

  • Mac上有brew，直接安装
shell
brew install ffmpeg
  • windows去官网下载

访问FFmpeg官方网站，在Windows部分，点击“Windows builds by btbn”或“Windows builds by zeranoe”（如果有可用）

js
Windows builds from gyan.dev
  • ‌下载FFmpeg‌：

下载的文件通常是一个zip压缩包。 ‌解压FFmpeg‌：将下载的文件解压到你选择的目录，例如c:\ffmpeg。

  • ‌配置环境变量‌： 右击“计算机”或“此电脑”，选择“属性”。 点击“高级系统设置”，然后点击“环境变量”。 在“系统变量”部分，找到并选中“Path”变量，点击“编辑”。 在“变量值”字段末尾添加FFmpeg的bin目录路径（例如c:\ffmpeg\bin），用分号分隔开。 点击“确定”保存更改。

打开命令提示符（按Win + R，输入cmd，然后按Enter）,检查FFmpeg是否安装成功：输入ffmpeg -version 并按Enter。

如果安装成功，会显示FFmpeg的版本信息。

2、使用 ffmpeg

ffmpeg是一个命令行工具，它允许你转换文件格式、过滤视频和音频流，以及录制、转换和流式传输音视频。

ffmpeg命令的基本语法是：ffmpeg [options] [[infile options] -i infile]... {[outfile options] outfile}...

常见操作示例：

  • ‌转换视频格式‌：ffmpeg -i input.mp4 output.avi
  • ‌提取音频‌：ffmpeg -i video.mp4 -q:a 0 -map a audio.mp3
  • ‌剪切视频‌：ffmpeg -i input.mp4 -ss 00:01:00 -to 00:02:00 -c copy output.mp4
  • ‌合并视频文件‌：创建一个文本文件（如filelist.txt），内容如下：file 'video1.mp4' file 'video2.mp4'，然后运行以下命令：ffmpeg -f concat -safe 0 -i filelist.txt -c copy output.mp4
