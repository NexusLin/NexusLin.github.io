❤ JS 变量 ​

主要是变量命名规则

js
JavaScript 标识符

所有 JavaScript *变量*必须以*唯一的名称*的*标识*。

这些唯一的名称称为*标识符*。

标识符可以是短名称（比如 x 和 y），或者更具描述性的名称（age、sum、totalVolume）。

构造变量名称（唯一标识符）的通用规则是：

-   名称可包含字母、数字、下划线和美元符号
-   名称必须以字母开头
-   名称也可以 `$` 和 `_` 开头（但是在本教程中我们不会这么做）
-   名称对大小写敏感（y 和 Y 是不同的变量）
-   保留字（比如 JavaScript 的关键词）无法用作变量名称

提示：JavaScript 标识符对大小写敏感。

slice() 方法 ​

slice() 提取字符串的某个部分并在新字符串中返回被提取的部分。

该方法设置两个参数：起始索引（开始位置），终止索引（结束位置）。 这个例子裁剪字符串中位置 7 到位置 13 的片段：

实例 ​
var str = "Apple, Banana, Mango";
var res = str.slice(7,13); //Banana
var res = str.slice(-13,-7); //Banana

省略第二个参数，则该方法将裁剪字符串的剩余部分

var res = str.slice(7); //Banana, Mango 【其中含有一个空格,格外注意】
var res = str.slice(-13); //Banana, Mango 从结尾截取

replace()方法用另一个值替换在字符串中指定的值： ​

js

str = "Please visit Microsoft and Microsoft!";
var n = str.replace("Microsoft", "W3School");
//Please visit W3School and Microsoft!
== replace() *只替换首个匹配*： ===  


`replace()` 对大小写敏感,需执行大小写不敏感的替换，请使用正则表达式 `/i`（大小写不敏感）

案列：

str = "Please visit Microsoft!";
var n = str.replace(/MICROSOFT/i, "W3School");
//结果输出  ：Please visit W3School!

== 需替换所有匹配，请使用正则表达式的 `g` 标志（用于全局搜索）==
==  正则表达式不带引号 ==

🍉把字符串转换为数组 ​

通过 split() 将字符串转换为数组

js

var txt = "a,b,c,d,e";   // 字符串
txt.split(",");          // 用逗号分隔
txt.split(" ");          // 用空格分隔
txt.split("|");          // 用竖线分隔


省略分隔符，被返回的数组将包含 index [0] 中的整个字符串。
如果分隔符是 ""，被返回的数组将是间隔单个字符的数组：

txt.split("");   // ['a', ',', 'b', ',', 'c', ',', 'd', ',', 'e']
txt.split(" ");   //  ['a,b,c,d,e']

## String.search()
`search()` 方法在字符串中搜索指定值并返回匹配的位置：



let str = "Please locate where 'locate' occurs!";
str.search("locate")     // 返回 7

indexOf() 和 search() 这两个方法并不相等。差别如下：

js
-   `search()` 方法不能接受第二个起始位置参数。
-   `indexOf()` 方法不能采用强大的搜索值（正则表达式）
  • String.match()

match() 方法根据正则表达式在字符串中搜索匹配项，并将匹配项作为 Array 对象返回。

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/g)    // 返回数组 [ain,ain,ain]

正则表达式不包含 g 修饰符（执行全局搜索），match() 方法将只返回字符串中的第一个匹配项

// ['ain', index: 5, input: 'The rain in SPAIN stays mainly in the plain', groups: undefined]

- 执行不区分大小写的全局搜索
let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/gi)   // 返回数组 [ain,AIN,ain,ain]
  • String.includes()
js

如果字符串包含指定值，`includes()` 方法返回 true。
string.includes(searchvalue, start)

start = 可选。默认为 0. 开始搜索的位置。
