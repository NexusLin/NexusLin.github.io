cmd命令行处理常见的git操作

通常我们的git操作如下：

JS
1. git init
2. git add .
3. git commit -m "first commit"

但是每次一遍一遍打包写✍实在太麻烦了，所以我们可以使用cmd脚本来处理这些操作。

新建一个文本文件,里面写入以下代码

JS
@echo off
 git_pull.bat - 自动执行 Git 操作的脚本

 执行 git pull
git pull
if %ERRORLEVEL% neq 0 (
    echo git pull failed, stopping...
    exit b %ERRORLEVEL%
)

 执行 git add .
git add .
if %ERRORLEVEL% neq 0 (
    echo git add failed, stopping...
    exit b %ERRORLEVEL%
)

 执行 git commit
git commit -m Your commit message
if %ERRORLEVEL% neq 0 (
    echo git commit failed, stopping...
    exit b %ERRORLEVEL%
)

echo Git commands executed successfully!

然后注意我们把上面的文件后缀改为.bat,这样我们就可以直接在cmd中使用了。

JS
git_auto.bat

优化一下，执行成功或者失败我们都不关闭窗口

JS
@echo off
:: 自动化 Git 操作脚本
:: 用于执行 git pull, git add, 和 git commit

echo Starting git pull...
git pull
if %ERRORLEVEL% neq 0 (
    echo "git pull failed, stopping..."
    pause
    exit /b %ERRORLEVEL%
)

echo Starting git add ...
git add .
if %ERRORLEVEL% neq 0 (
    echo "git add failed, stopping..."
    pause
    exit /b %ERRORLEVEL%
)

echo Starting git commit...
git commit -m "Your commit message"
if %ERRORLEVEL% neq 0 (
    echo "git commit failed, stopping..."
    pause
    exit /b %ERRORLEVEL%
)

echo "Git commands executed successfully!"
pause

☞ Linux版本 ​

Linux 中，脚本通常是用 Bash 编写的

JS
#!/bin/bash
# 自动化 Git 操作脚本
# 用于执行 git pull, git add, git commit 和 git push

# 捕获 Ctrl+C 中断
trap 'echo "Aborted!"; exit' INT

echo "Starting git pull..."
git pull
if [ $? -ne 0 ]; then
    echo "git pull failed, stopping..."
    exit 1
fi
echo "Git pull completed successfully."

echo "Starting git add..."
git add .
if [ $? -ne 0 ]; then
    echo "git add failed, stopping..."
    exit 1
fi
echo "Git add completed successfully."

echo "Starting git commit..."
git commit -m "Your commit message"
if [ $? -ne 0 ]; then
    echo "git commit failed, stopping..."
    exit 1
fi
echo "Git commit completed successfully."

echo "Starting git push..."
git push
if [ $? -ne 0 ]; then
    echo "git push failed, stopping..."
    exit 1
fi
echo "Git push completed successfully."

# 保持窗口打开
echo "All Git commands executed successfully!"
echo "Press Ctrl+C to close the terminal."
# 让脚本结束后仍保持终端打开
bash

数字雨cmd ​

JS
@echo off  
:line 
color 0a
setlocal ENABLEDELAYEDEXPANSION  
 
for /l %%i in (0) do (  
set "line="  
for /l %%j in (1,1,80) do (  
set /a Down%%j-=2  
set "x=!Down%%j!"  
if !x! LSS 0 (  
set /a Arrow%%j=!random!%%3  
set /a Down%%j=!random!%%15+10  
)  
set "x=!Arrow%%j!"  
if "!x!" == "2" (  
set "line=!line!!random:~-1! "  
) else (set "line=!line! ")  
)  
set /p=!line!<nul  
)
goto line
