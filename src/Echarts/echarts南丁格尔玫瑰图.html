3、🍉详细参数配置 ​

🍓 给每个圈配置自定义颜色 ​

option => color

js
颜色设置在option 对象里面外层直接写一个数组
color: ['#1A83FF', '#5DA0FF', '#8AB7FF', '#AFCEFF', '#D1E3FF'],

直接使用活数据颜色 ​

option => series => data => itemStyle => color

js
option = {
  series: [{
    type: 'pie',
    radius: ['40%', '60%'], // 设置为圆环图
    data: [
      { value: 100, name: '苹果', itemStyle: { color: '#ff7f50' } },  // 设置苹果项的颜色
      { value: 200, name: '香蕉', itemStyle: { color: '#ffcc00' } },  // 设置香蕉项的颜色
      { value: 150, name: '橙子', itemStyle: { color: '#ffa500' } }   // 设置橙子项的颜色
    ]
  }]
};

🍓 更改圈的大小 ​

调整option下series里面的redius【也可以借这个参数实现不同的内圈外圈和负向向里】

🍓扇形图上文字展示位置 ​

在option => series=> label 对位置进行设置

js
 label:{
   show:true,
   position:'inside', 
   //inside 扇形图上 ，center 扇形图中间 ,  outside  扇形图外面
},

然后我们看看效果

🍓扇形图线段大小（outside时生效） ​

在option => series=> label=> distance 对位置进行设置

js
 label: {
    show: true,
    position: 'outside', // 标签显示在外部
    distance: 15, // 标签与扇形图形之间的距离
    formatter: '{b}: {c}' // 标签格式
  }

🍓更改显示文字名称和值以及换行 ​

在option => series=> label=> formatter设置

js
 label:{
   show:true,
   position:'inside', 
   formatter: '{b}:{c}:{d}%',
   //inside 扇形图上 ，center 扇形图中间 ,  outside  扇形图外面
},

踩坑 ​

这里格外注意formatter之中的换行br是不生效的

如果你有换行的需求，应该已经尝试了很多次了就是换行不生效，这是因为这里得用\n,哈哈

js
 label:{
   show:true,
   position:'inside', 
   formatter: '{b}:\n{c}:\n{d}%',
   //inside 扇形图上 ，center 扇形图中间 ,  outside  扇形图外面
},

效果:

更改文字大小和颜色 ​

在option下series下label 里面添加一些具体参数

js
  position: 'center', // 数值显示在内部
  fontSize: 18,
  color: '#1890FF',
  fontWeight: 600, 
  backgroundColor:'#fff', //
  width:68,

⑤ 修改提示文字的格式

在option下series下label 里面修改参数formatter

js
  formatter: '{d}%', // 格式化数值百分比输出
